## GIT COMMAND RECOMMENDATIONS

### For version control

1. `git add 'path/to/file'`

2. `git commit -m '$1'`

3. `git pull --rebase`

4. `git push`

### For cloning

1. `git clone --recurse-submodules URL/SSH`

2. `cd 'repo-name'`

3. `git lfs install` (make sure git lfs is installed on system)

4. `git lfs pull`
