#Sumofthree.py

Först så ber jag användaren om ett tre siffriga nummer.
Sen så delar jag in siffrorna i 3 variabler med hjälp av % så plockar jag ut den sista siffran och sedan tar bort den från input variablen med hjälp av // 
Efter detta så beräknar jag summan av dessa 3 variabler som nu representerar inputen.
Slutligen så printar jag ut resultatet för användaren.

#Change.py

Nu så börjar jag med att kolla om användaren ska ha några tusenlappar tillbaka och hur många isåfall med hjälp av // sedan upprepar jag detta för resterande sedlar och mynt och skriver ut resultatet till användaren
